Catch N Match

Description:

Catch-N-Match is an exciting event which caters you with Fun,Funda and Fest.

The main motto of this event is to make the participants to test their technical skills without a single pie.In this event, a code will be divided into multiple snippets and you have to catch those code snippets from various places by using clues and assemble them in one place and execute it.

Rules and Regulations:

1. It’s a team event.

2. Each team size must be 3.

3. Each team member must be registered.

4. There will be two rounds in this event.

5. Round-1 meant for qualifier round and Round –2 meant for final round.

6. During the contest switching on mobiles is prohibited. No more internet surfing.

7. Any compensatory actions will not be allowed.

8. No extra time will be allocated to any of the paticipant/team.
